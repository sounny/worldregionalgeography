# Exams

This folder contains the Midterm and Final Exam content for Canvas.

## Exam Overview

| Exam    | Chapters Covered                              | Week                  | Duration   | Points |
| ------- | --------------------------------------------- | --------------------- | ---------- | ------ |
| Midterm | 1-6 (Introduction through Sub-Saharan Africa) | Week 8 (March 2-6)    | 60 minutes | 100    |
| Final   | All (1-12), emphasis on 7-12                  | Finals Week (May 2-7) | 90 minutes | 150    |

## Exam Format

- **Question Types**: Multiple Choice, True/False, Matching, Map Identification
- **Auto-Graded**: Yes (Canvas)
- **Attempts**: 1
- **Proctoring**: [Specify if using Respondus Lockdown Browser or Honorlock]

## Files

- `midterm_exam.md` - Midterm exam questions and answer key
- `final_exam.md` - Final exam questions and answer key
- `map_identification_bank.md` - Map-based questions for both exams

## Canvas Settings Recommendations

- Enable "Shuffle Questions" to randomize order
- Enable "Shuffle Answers" for multiple choice
- Show correct answers only after availability ends
- Require Respondus LockDown Browser (optional, for high-stakes proctoring)
