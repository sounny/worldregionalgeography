# Weekly Assignments (Auto-Graded Quizzes)

This folder contains the weekly quiz content for Canvas import. Each quiz is designed to be auto-graded and based on the corresponding chapter in the textbook.

## Quiz Format

- **Questions**: 10-15 per quiz
- **Question Types**: Multiple Choice, True/False, Matching
- **Time Limit**: 30 minutes
- **Attempts**: 1
- **Due**: Sunday at 11:59 PM each week
